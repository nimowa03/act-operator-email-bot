Problem

AI가 이메일을 마음대로 보낸다면?

"김팀장에게 회의록 보내줘" → AI가 즉시 전송

Act Operator

명령어 4개로 끝나는 AI 프로젝트

명령어역할소요
act new프로젝트 뼈대 생성10초
uv sync의존성 설치5초
act cast새 워크플로우 추가5초
langgraph dev시각적 디버깅 서버3초
🎬

Act = 영화 한 편 (전체 프로젝트)  |  Cast = 배우 (개별 워크플로우). 하나의 Act 안에 여러 Cast가 독립 패키지로 존재합니다.

interrupt() 한 줄이 HITL의 핵심. 그래프 실행을 완전히 멈추고, Command(resume="yes")로 재개합니다. Checkpointer가 중단 상태를 저장합니다.

Code

interrupt() 한 줄의 힘

class ReviewEmailNode(BaseNode):

    def execute(self, state):
        review_info = f"수신자: {state['email_to']}\n제목: {state['email_subject']}"

        # ⭐ 이 한 줄이 그래프를 멈춘다
        decision = interrupt(review_info)

        # 사용자가 resume하면 여기부터 이어서 실행
        if decision == "yes":
            return {"messages": [AIMessage("승인됨")]}
💾

Checkpointer = 게임의 세이브 포인트. interrupt()로 멈추면 상태가 저장되고, resume할 때 저장된 상태에서 이어서 실행됩니다.

Live Demo

실제 Gmail 전송 시연

# Step 1: 이메일 작성 요청
graph.invoke({"query": "makegrowth101@gmail.com에게 테스트 메일 보내줘"})

⏸️ === 이메일 승인 요청 ===
수신자: makegrowth101@gmail.com
제목: 테스트 이메일
승인하려면 'yes', 거부하려면 'no'

# Step 2: 사용자 승인
graph.invoke(Command(resume="yes"))

✅ 이메일 전송 완료: makegrowth101@gmail.com
